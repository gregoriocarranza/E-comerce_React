## > Instalacion

1. Descarga el repositorio

2. En la carpeta de descarga ( y con la consola abierta) ingresa el siguiente comando

   ```console
   npm install
   ```

3. Luego ejecuta el comando listado para iniciar la pagina

   ```console
   npm start

   ```
